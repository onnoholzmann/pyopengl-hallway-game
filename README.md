# Hallway 3d game

This is an arcade style game about a player, who is trapped in a game. You need to try to get as many points as possible before the glitch consumes you. You start in a simple platformer, where you need to collect as many coins as possible.
![A coin and a gold bag in the platformer game](/readme_images/platformer.png)
You will glitch into the main game, where you are trapped in a maze of hallways, where you need to collect as many blocks before the glitch fully consumes you.
![A block in the hallway](/readme_images/hallway.png)
You can try to improve your score and get a higher highscore.
![The score board](/readme_images/score.png)

## **Dependencies**

1. **python 3.12+**(recommended install via apt, pacman or the microsoft store)
2. use something like **yay** or **pip** to install the **packages**
    1. pygame
    2. pyopengl
    3. pyopengl-accelerate
    4. numpy
    5. copy
   ~~~
   pip install pygame pyopengl pyopengl-accelerate numpy
   ~~~

## **Run**

In the terminal, run in the hallway_3d directory:
~~~
python3 main.py
~~~
